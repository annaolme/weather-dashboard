# Weather Dashboard

Search for weather by city name using the OpenWeatherMap API.

## Features

- Search any city
- Shows temperature, humidity, wind speed
- Weather icons
- Save favorite cities (stored in localStorage)
- Responsive

## Setup

1. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
2. Replace `YOUR_API_KEY_HERE` in `app.js`
3. Open `index.html`
